Haunted House – First Person Horror Game

A first-person horror game prototype developed in Unity 3D.

Description

This project is a prototype of a first-person horror game set inside a haunted house. The player explores the environment, interacts with objects, and experiences scripted horror events triggered throughout the level.

The main goal of this prototype was to implement and test core gameplay mechanics such as movement, interaction systems, and environmental events.

Current Features

The prototype currently includes:

First-person player movement

Object interaction system

Object pickup mechanics

Torch mechanic to burn spider webs

3D haunted house environment

Trigger-based jumpscares

Basic environment exploration

Implemented Mechanics
Player Movement

The player can freely move through the environment using a first-person controller.

Object Interaction

Players can interact with and pick up objects placed in the environment.

Torch and Spider Webs

The player can use a torch to burn spider webs, allowing access to new areas in the house.

Jumpscares

The environment includes trigger zones that activate scripted jumpscare events when the player enters specific areas.

Technologies Used

Unity 3D

C#

Collider and trigger systems

3D environment assets

Project Status

This project is currently a gameplay prototype.
Core mechanics are functional, but additional systems are planned for future development, such as:

Inventory system

Enemy AI

Advanced sound design

Narrative elements and objectives

Lighting and visual improvements

Project Goal

The goal of this project is to develop a functional horror game prototype, focusing on gameplay mechanics before expanding into a complete game experience.
